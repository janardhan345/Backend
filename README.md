# This Repo contains a Basic Backend project to understand the fundamentals of Backend
##  Node.js Authentication & CRUD API

This project is a backend REST API built to practice core backend concepts
such as CRUD operations, authentication, and secure password handling.

The API is built using Node.js, Express.js, and MongoDB with Mongoose.

## What I Learned

- Building REST APIs using Express.js
- Implementing CRUD operations
- Authentication and password hashing using bcrypt
- Environment variable management using dotenv
- MongoDB data modeling with Mongoose

## Features

- User Registration
- User Login
- Secure Password Hashing
- Create Data
- Read Data
- Update Data
- Delete Data
---
## Installation

1. Clone the repository

git clone https://github.com/yourusername/project-name.git

2. Install dependencies

npm install

3. Create a .env file

PORT=5000
MONGO_URI=your_mongodb_connection_string

4. Start the server

npm run dev
--- 
## Dependencies 

- Express -> route, middleware API endpoints 
- Mongoose -> to interact with schemas and models 
- dotenv -> to load variables from .env file to process.env
- bcrypt -> hashes the password

## Dev Dependencies  
- nodemon -> automatically updates server when a change is detected.
